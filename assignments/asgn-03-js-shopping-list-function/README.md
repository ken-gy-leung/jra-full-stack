## 作业目标:

通过这次的作业，你将会练习JavaScript中的基本数据类型、数组、对象、条件语句、循环语句和函数的应用。

1. 创建一个购物清单数组，例如:
  
![Shopping-List-Array](README-imgs/Shopping-List-Array.png)
